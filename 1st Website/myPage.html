Tags

The language of HTML revolves around tags. Every component in a website is
contained within a tag. The first tag you create on a website declares the doctype,
or the type of file your document will be.

<p> : This is the paragraph tag. Whenever you want to have writing in your website
you put this before the paragraph and then you put this at the end of the paragraph: </p>

<br/> : This is the line break tag. HTML can't read line breaks by itself so whenever you want a
line break to be put somewhere you have to put this tag. Here is some tags in an unordered list:

  • br
  • p
  • hr
  • li
Here is an ordered list of my top 3 favorite tags:
  1. p
  2. br
  3. hr

Links

No website in complete without having some links to other websites. In order to add links to your website you use
the href tag. The format used in HTML for using links looks like this: <a href = "link goes in quotations"> what you want the link to say </a>.
